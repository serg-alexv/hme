# Architecture Decision Records

Every material architecture, schema, platform, security, cost, or claims decision receives an ADR.

## File naming

```text
ADR-0001-short-decision-title.md
```

## Required structure

```markdown
# ADR-0000: Title

- Status: proposed | accepted | superseded | rejected | deferred
- Date: YYYY-MM-DD
- Owner:
- Decision class: A | B | C | D

## Context
## Observed facts
## Options considered
## Decision
## Consequences
## Evidence
## Risks
## Reversal / reactivation gate
## Claims impact
## Y/N closure
```

ADRs describe decisions, not post-hoc mythology. Uncertainty and rejected alternatives must remain visible.
